git PUSH setting
-----------------
want to check your setting, do the following
git config --global push.default

To push the current branch and set the remote as upstream, use
git push --set-upstream origin master

To push the current branch to a branch of the same name.
git config --global push.default current

to keep track of commit and stay up-to-date
git push -u
